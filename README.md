# Sales Performance Dashboard

## Project Overview
Designed and developed an interactive, executive-level Sales Performance Dashboard using Excel (WPS Office). This project transforms raw retail transactional data into a dynamic visual analytics platform to track global revenue, profitability, and distribution trends.

## Key Features & Insights
**Executive Performance Tracking:** Monitored vital metrics including **$12,642,502 in Total Sales**, **$1,467,457 in Total Profit**, **51,290 Total Orders**, and an **11.6% Profit Margin**.
**Multidimensional Data Visualization:** Built five dedicated reporting views:
**Sales by Region:** Multi-region tracking spanning Africa, Canada, Caribbean, Central, Central Asia, East, EMEA, North, North Asia, Oceania, South, Southeast Asia, and West.
**Monthly Sales Trend:** A continuous timeline chart mapping seasonal sales distribution from January to December.
**Profit by Sub-Category:** Horizontal bar chart highlighting profitability performance across product lines (Supplies, Paper, Furnishings, Copiers, Binders, Accessories, etc.).
**Ship Mode Distribution:** A breakdown of fulfillment methods mapping Standard Class, Second Class, First Class, and Same Day shipping volumes.
**Interactive Self-Service BI:** Implemented **dynamic slicers (Year, Category, Region, Segment)** allowing stakeholders to drill down seamlessly into specific data subsets.


## Technical Tool Stack
* **Software:** Microsoft Excel / WPS Office
* **Features Used:** Pivot Tables, Pivot Charts, Slicers, Custom KPI Cards, and Advanced Data Formatting.
